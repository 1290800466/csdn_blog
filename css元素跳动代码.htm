代码示例：

<span style="font-size:18px;"><!DOCTYPE html>
<html>
<head>
<meta charset="utf-8">
<style>
.scroll{
	position:absolute;
	animation:1s arrow infinite ease;
}
@keyframes arrow {
	0%,100% {
		top:50px;
	}
	50% {
		top:200px;
	}
}
</style>
</head>
<body>
<b class="scroll">滑动鼠标滚轮</b>	
</body>
</html></span>


以上代码会控制<b> 标签中元素上下跳动，若将 top  改为 left  则是左右跳动。
