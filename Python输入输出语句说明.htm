print输出语句说明：

同时输出多个值：printx,t,n

Print格式化输出：

（1）小数值确定精度输出

print(format(值，‘格式字f’))   如：输出12.34567  保留3位小数（注意此时是四舍五入）且输出占6位

Print(format(12.34567,’6.3f’))


12.346

当保留小数位多于小数位数时，后面用0补足；

当输出位数大于实际位数时，前面用空格补足，即右对齐显示；

当输出位数小于实际位数时，对输出无影响，左对齐显示；

（2）小数值转化为百分数输出

Print(format(0.345678,’.3%’))
#34.568%
Print(format(0.345678,’4.3%’))
#34.568%
Print(format(0.345678,’9.3%’))
#34.568%


raw_input()输入语句说明：

注意raw_input()函数读取的数值都作为字符串类型数据

若想输入的字符为其他类型，可以;

 age= int(raw_input("your age is:"))

此时读取的数据为int型字符
