HTML框架示例1:

<html>

<head>
<title>HTML框架示例</title>

</head>

<!--各窗口的尺寸设置，
cols表示纵向排列多个窗口：
rows表示横向排列多个窗口-->
<frameset  cols = "80%,20%">

<!--默认情况下排列好的窗口可以用鼠标移动，若在其后加 noresize 属性，则排列好后不能移动-->
<!--frameborder属性确定有无边框，0表示无，1表示有边框-->
<!--name 对窗口进行标识，之后可对此标识进行各窗口间相互操作（起名，以便进行替换）-->
<frame src = "歌曲列表.html" name = "a1" noresize >

<frame src = "b.html"  name = "a2" frameborder = "0">

</frameset>

</html>

注意：如果a.html包括了其他的页面（frameset）则要求a.html本身不能有body模块(不能出现body标记，代码中也不能出现body模块中才有的内容，如文字等)。


如何去掉Html中frameset产生的横向滚动条

在body标签里写上style="OVERFLOW:SCROLL;OVERFLOW-X:HIDDEN"

同理去竖条的话就是把OVERFLOW-X:HIDDEN改成OVERFLOW-y:HIDDEN

如在一下的  歌曲列表.html  的body标签中加上该style

歌曲列表.html 

<html>
<head>
<meta http-equiv="Content-Type" content="text/html; charset=gb2312">
<title>歌曲列表</title>
</head>
<p><body <span style="font-family: Verdana, Arial, Helvetica, sans-serif; font-size: 13px; line-height: 19.5px; background-color: rgb(254, 254, 242);">style="OVERFLOW:SCROLL;OVERFLOW-X:HIDDEN"</span>></p><!--target表示我们点击后，目标指向谁（也就是在哪显示）-->
<a href = "周杰伦.html" target="a2">周杰伦</a><br>
<a href = "汪峰.html" target="a2">汪峰</a><br>
</body>
</html>

Ø  Target属性有四个属性值：

Ø  <a href=url  target=_blank> 在新窗口打开
<a href=url  target=_self> 在本窗口打开
<a href=url  target=_parent> 父窗口
<a href=url  target=_top> 整个浏览器窗口

Ø  还有一种就是在target后直接写上要替换的frame的名字
