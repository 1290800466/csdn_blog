Python下使用第三方函数库：

以httplib2 为例

以下代码为读取某网页信息：

>>> import urllib    //该库可以访问web网页

>>>url="http://www.163.com"                  

>>>content=urllib.urlopen(url).read()   //读取某网页信息

>>>open("163.com.html","w").write(content)    //将读取的网页信息写入163.com.html文件，若没有该文件则新建一个。

 

 

>>> import webbrowser                    //该库用来调用本地浏览器

>>>webbrowser.open_new_tab("163.com.html")   //将163.com.html文件用浏览器打开

>>> webbrowser.open_new_tab("www.163.com")

 

>>> import webbrowser as web  //该库用来调用本地浏览器,对于较长的模块名，为方便，可以给他起一个假名，如webbrowser 起假名web。Python语句基本语法：

Module_name.method(parameters)。

>>> web.browser.open_new_tab("163.com.html")   //将163.com.html文件用浏览器打开

>>> web.browser.open_new_tab("www.163.com")
